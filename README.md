# Lisp-Interpreter

Create venv:
```shell
python -m venv .venv
```

Install dependencies:
```shell
pip install -r requirements.txt
```

Format code with black:
```shell
black .
```

Check the code quality with pylint:
```shell
pylint .
```

Launch code via cli with:
```shell
python main.py <path>
```
